# Ringkasan perubahan Klinik Pintar

Dokumen ini merangkum perbedaan utama antara versi dasar aplikasi dan versi update yang sudah dirapikan untuk dipublikasikan ke GitHub.

## Perubahan utama

### 1. Data poli per cabang
- Menambahkan menu khusus untuk mengelola poli.
- Mendukung tambah, ubah, nonaktifkan, dan aktifkan kembali poli.
- Pengelolaan mengikuti cabang aktif.

### 2. Rekam medis yang lebih rinci
- Menambahkan profil medis pasien.
- Menambahkan catatan medis per kunjungan.
- Menambahkan monitoring pasien kontrol rutin.
- Menampilkan timeline riwayat medis pasien.

### 3. Profil user mandiri
- User dapat mengubah data dirinya sendiri.
- User dapat ganti password.
- User dapat upload, ganti, dan hapus foto profil.
- Avatar default otomatis mengikuti role bila belum ada foto pribadi.

### 4. Dashboard spesifik per role
- Dashboard dokter fokus pada pasien, pemeriksaan, dan kontrol.
- Dashboard kasir fokus pada tagihan, pembayaran, dan pendapatan.
- Dashboard farmasi fokus pada resep dan stok obat.
- Dashboard inventory fokus pada stok dan restok.
- Dashboard front office fokus pada pendaftaran dan antrian.
- Dashboard manajemen fokus pada ringkasan operasional cabang.

### 5. Penyegaran antarmuka
- Tampilan dibuat lebih modern dan konsisten.
- Perbaikan kontras warna agar teks lebih mudah dibaca.
- Penghapusan keterangan yang tidak perlu agar tampilan lebih bersih.
- Avatar role dibuat lebih rapi dan terpusat.

### 6. Standarisasi format data
- Nominal uang diformat ke Rupiah.
- Nomor telepon dirapikan ke pola XXXX-XXXX-XXXX.

## Sebelum dan sesudah

### Sebelum
- Dashboard masih sama untuk banyak role.
- Belum ada modul rekam medis rinci.
- Belum ada menu poli per cabang.
- Profil user masih terbatas.
- Format uang dan telepon belum konsisten.
- Tampilan masih belum seragam di banyak halaman.

### Sesudah
- Dashboard berbeda sesuai kebutuhan tiap role.
- Rekam medis pasien lebih lengkap.
- Ada pengelolaan poli per cabang.
- User dapat mengelola profil sendiri.
- Format data lebih rapi dan konsisten.
- Antarmuka menjadi lebih modern dan bersih.
